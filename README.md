# 소수결 게임 Ultimate v11

v11은 **GitHub 없이, BAT 파일 없이 업데이트하기 쉬운 구조**를 목표로 정리한 버전입니다.

## 핵심
- GitHub 필요 없음
- BAT 파일 필요 없음
- Node.js 필요 없음
- 기존 폴더에 파일 덮어쓰기 방식 가능
- UPDATE_ONLY_v11 폴더 제공
- config 폴더 보존 구조
- 교사용 화면에 브라우저 설정 백업/복원 추가

## 기능
- 개인전 / 팀전
- 학생 직접 조 선택 / 랜덤 균형 배정
- 교사 조 변경
- 학생은 라운드 종료 전까지 선택 변경 가능
- 보기 3개 개별 입력
- 한글 이름 입력 안정화
- 전자칠판 화면
- 선택 비율 그래프
- 개인/팀 랭킹
- CSV 저장

## 주소 유지 원칙
주소를 유지하려면 새 Vercel 프로젝트를 만들면 안 됩니다.
기존 프로젝트에 다시 배포해야 합니다.
