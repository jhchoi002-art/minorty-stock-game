소수결 게임 Ultimate v10 배포/업데이트 안내

[가장 쉬운 방법: Vercel 웹 업로드]
1. ZIP 파일을 압축 해제합니다.
2. Vercel 프로젝트로 들어갑니다.
3. 새 배포(Create Deployment/Upload)에서 압축 해제한 폴더를 올립니다.
4. Framework: Other / Static
5. Build Command: 비워두기
6. Output Directory: . 또는 비워두기

[GitHub 없이 명령어로 업데이트]
1. Node.js LTS 설치
2. 이 폴더에서 Deploy_Update.bat 실행
3. 처음 실행 시 Vercel 로그인을 진행합니다.
4. 이후에는 같은 폴더에서 Deploy_Update.bat만 실행하면 됩니다.

주의:
- 학교 PC 보안 정책에 따라 명령어 실행이 막힐 수 있습니다.
- 그럴 경우 Vercel 웹 업로드 방식이 가장 안전합니다.
