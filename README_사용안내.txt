소수결 주식 게임 v8 수정판

반영 내용
1. 미제출 학생이 있을 때 새 라운드 시작 경고
- 라운드 진행 중 미제출자가 남아 있는 상태에서 '새 라운드 시작'을 누르면 확인창이 뜹니다.
- 확인을 눌러야 새 라운드가 시작됩니다.

2. 학생 재입장 이어하기
- 학생이 실수로 나가도 같은 방 코드 + 같은 이름으로 다시 들어오면 기존 학생 기록으로 이어집니다.
- 기존 총점, 마지막 점수, 현재 라운드 선택 상태가 유지됩니다.

업데이트 방법
1. 이 ZIP 파일 압축 풀기
2. GitHub 저장소에서 Add file → Upload files
3. 압축 푼 폴더 안의 파일 전체를 드래그해서 덮어쓰기
4. Commit changes 클릭
5. Vercel이 자동으로 다시 배포합니다.
