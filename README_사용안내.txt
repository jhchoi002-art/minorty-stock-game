소수결 게임 Ultimate 9.0

완전 재설계 버전입니다.

핵심 안정화
- 보기 입력을 textarea 줄바꿈 방식에서 보기 1/2/3 개별 입력칸으로 변경
- '보기1\n보기2\n보기3'처럼 깨지는 문제 방지
- 학생 화면에서 보기가 섞이지 않도록 options 배열을 직접 저장

기능
- 방 만들기 / 방코드 자동 생성
- 학생 QR 접속
- 교사용 화면
- 전자칠판 화면
- 개인전 / 팀전
- 팀전일 때만 조 개수 활성화
- 학생 직접 조 선택 / 랜덤 균형 배정
- 교사 조 이동
- 라운드 종료 전 학생 선택 변경 가능
- 선택 비율 그래프
- 개인/팀 랭킹
- CSV 저장
- 300문제 내장

배포
- 정적 사이트입니다.
- npm install 필요 없음
- Vercel에서 폴더 업로드
- Framework: Other / Static
- Build Command: 비워두기
- Output Directory: 비워두기 또는 .

Firebase Rules
{
  "rules": {
    ".read": true,
    ".write": true
  }
}


v9.1 수정사항
- 학생 화면에서 이름/번호 입력 시 한글 자음·모음 입력마다 화면이 다시 렌더링되어 입력이 끊기던 문제를 수정했습니다.
- 이름 입력 중에는 화면을 다시 그리지 않고, 선택 버튼을 눌렀을 때 이름을 확인하도록 변경했습니다.
- 랜덤 팀 배정은 이름 입력 후 보기를 누르면 자동 배정되도록 안정화했습니다.
