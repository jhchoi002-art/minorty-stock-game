소수결 주식 게임 v8 실험판

기준 버전: v7 선택 수정 가능 버전

변경 사항
1. 학생이 A/B/C를 선택하거나 바꿔도 효과음이 나지 않습니다.
2. 학생 선택/변경 시 알림창이 뜨지 않습니다.
3. 갤럭시 탭 S6 Lite 가로모드에 맞게 학생 화면을 크게 조정했습니다.
4. A/B/C 선택 버튼이 가로 3분할 큰 버튼으로 보입니다.
5. 학생은 교사가 라운드를 마감하기 전까지 선택을 계속 수정할 수 있습니다.
6. v8 전용 Firebase 경로(stockRoomsV8)를 사용하므로 v7 방과 기록이 섞이지 않습니다.
7. 학생 입장 시 방 코드가 실제로 존재하는지 확인합니다.
