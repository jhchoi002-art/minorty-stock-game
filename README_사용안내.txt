소수결 주식 게임 v7 - 선택 수정 가능 버전

1. ZIP 압축을 풉니다.
2. GitHub 저장소에 index.html, app.js, style.css 파일을 업로드합니다.
3. Vercel에서 해당 GitHub 저장소를 배포합니다.

주요 기능
- 기존 v6 방식처럼 Firebase Realtime Database를 코드 안에서 사용합니다.
- Redis / Upstash / 환경변수 설정은 필요 없습니다.
- 교사가 방을 만들고 학생은 QR로 입장합니다.
- 학생이 A/B/C를 선택하면 교사 화면에 자동 반영됩니다.
- 학생은 라운드 종료 전까지 선택을 다시 바꿀 수 있습니다.
- 교사가 라운드 마감 버튼을 누르면 그 시점의 마지막 선택으로 점수가 계산됩니다.
- 이벤트 배율 n배 입력 가능
- 라운드별 ABC 점수 그래프, 순위표, CSV 저장 포함


[수정 사항]
- 학생이 A/B/C를 선택하거나 바꿀 때 효과음이 나지 않도록 수정했습니다.
- 선택 저장 알림창을 제거하여 수업 중 조용히 진행되도록 했습니다.
- 교사가 라운드를 종료하기 전까지 학생은 계속 선택을 바꿀 수 있습니다.
