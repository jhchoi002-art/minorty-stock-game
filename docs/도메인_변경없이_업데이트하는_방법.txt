소수결 게임 Ultimate v11
도메인 변경 없이 업데이트하는 자세한 방법

[핵심 원칙]
주소가 바뀌는 이유는 매번 Vercel에서 새 프로젝트를 만들었기 때문입니다.
앞으로는 기존 프로젝트 하나만 계속 사용해야 합니다.

예:
기존 프로젝트: minority_game_v6_static_final
기존 주소: https://minority-game-v6-static-final.vercel.app

이 프로젝트를 계속 사용하면 v10, v11, v12로 바뀌어도 주소는 그대로 유지됩니다.

[업데이트 파일 덮어쓰기]
1. v11 ZIP을 압축 해제합니다.
2. UPDATE_ONLY_v11 폴더를 엽니다.
3. 그 안의 파일과 js 폴더를 모두 선택합니다.
4. 기존 소수결 게임 배포용 폴더에 붙여넣기 합니다.
5. 같은 이름의 파일을 바꾸겠냐고 물으면 '예'를 누릅니다.
6. config 폴더는 삭제하지 않습니다.

[가장 중요한 주의]
Vercel에서 New Project를 누르면 새 주소가 생깁니다.
주소를 유지하려면 기존 프로젝트에 다시 배포해야 합니다.

[Vercel 최신 UI 문제]
Vercel 최신 화면에서는 기존 프로젝트에 직접 폴더 업로드 버튼이 안 보일 수 있습니다.
이 경우 GitHub/CLI 없이 완전히 같은 주소로 덮어 배포하는 기능이 제한될 수 있습니다.
그래도 주소를 유지하려면 반드시 같은 Vercel 프로젝트를 기준으로 관리해야 합니다.

[현실적인 운영 추천]
1. 고정으로 사용할 Vercel 프로젝트 하나를 정합니다.
2. 인디스쿨에 올릴 주소도 그 주소 하나만 씁니다.
3. 업데이트할 때마다 새 프로젝트를 만들지 않습니다.
4. UPDATE_ONLY_v11 파일을 기존 폴더에 덮어씁니다.
5. Vercel에서 같은 프로젝트로 배포합니다.

[만약 새 프로젝트로 올릴 수밖에 없다면]
기존 vercel.app 주소는 그대로 유지하기 어렵습니다.
이 경우에는 나중에 Vercel의 Domains 메뉴에서 별도 도메인을 연결하는 방식이 필요합니다.
